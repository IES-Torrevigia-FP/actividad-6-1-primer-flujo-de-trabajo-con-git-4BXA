¿Qué diferencia hay entre estar “untracked” y estar en “staging”?

La principal diferencia entre estar "untracked" (sin seguimiento) y en "staging" (área de preparación) radica en si Git conoce el archivo y si este ha sido marcado para la próxima confirmación (commit).


¿Qué comando has utilizado para pasar el archivo a la zona de staging?

git add <nombre-del-archivo>
